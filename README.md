# TREASURE_HUNT_GAME
Treasure Hunt Game is a simple Python console game where the player searches rooms to find a hidden treasure. The treasure is randomly placed, and the player has three lives to guess correctly. It helps beginners practice loops, conditions, input handling, and the random module while enjoying interactive gameplay.
